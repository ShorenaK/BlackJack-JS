Game Descriptions:

This is the card game called "BlackJack", played at the table of 2-7 players against the dealer. The main goal of the game is to get to the value of 21 which is the "BlackJack". In my game only the dealer and one player play against each other,  the game starts by clicking the "Kick off the game" button, two cards per each are shuffled, and the aim is to beat the dealer by getting a score as close as possible to 21 or even to 21 it's self. After  cards are dealt player has an option to draw a new card, I have created the new card button so that the player can generate another set of cards if the  "New Card" button is clicked and the sum of the cards is over 21 one, player will have a message displayed accordingly "Bust! You lost a Bet!", if player gets number 21 a message  will appear "Player Wins Black Jack!!!", in addition player gets 50 chips added to the total of chips that player has. The player starts with 1000 chips and if the player loses, 50 chips will be deducted from the total, and in case of winnings 50 chips will be added. The value of the cards: from 2 to 10 worth their face value. Jack, Queen, and King are also worth 10, and ace is defaulted to 11 or it can be changed to one. In my game I have a slightly different setup since I only have numbers displayed, I have implemented the function that generates numbers from 1 to 11 if a player and dealer get the same value  of cards, a message will appear stating that " it's TIE", in case of the dealer hitting the black Jack, the player automatically loses, plus 50 chips will be deducted and message "'Table Wins Black Jack!!!" will be displayed. This game also has a third button to navigate to the main page which is the entrance of the casino the starting point to enter Shorena's Black Jack Table!
